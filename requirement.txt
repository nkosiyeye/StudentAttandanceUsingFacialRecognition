**Instructions to Set Up a Flask Project in VSCode**

### Step 1: Extract the Project
1. Download the zipped project file.
2. Extract it to your desired directory.
3. Open **Visual Studio Code**, then click **File > Open Folder** and select the extracted project folder.

### Step 2: Create and Activate a Virtual Environment
#### On Windows:
1. Create a virtual environment:
   ```sh
   python -m venv .venv
   ```
2. Activate the virtual environment:
   ```sh
   .venv\Scripts\activate
   ```

#### On macOS/Linux:
1. Create a virtual environment:
   ```sh
   python3 -m venv .venv
   ```
2. Activate the virtual environment:
   ```sh
   source .venv/bin/activate
   ```

### Step 3: Select Python Interpreter in VSCode
1. Open **VSCode**.
2. Press **Ctrl + Shift + P** to open the command palette.
3. Search for **Python: Select Interpreter** and select it.
4. Choose the interpreter inside your virtual environment:
   - On Windows: `.venv\Scripts\python.exe`
   - On macOS/Linux: `.venv/bin/python`

### Step 4: Install Dependencies
1. Ensure your virtual environment is activated.
2. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```

### Step 5: Run the Flask Application
1. Set the Flask application environment variable:
   - On Windows (Command Prompt):
     ```sh
     set FLASK_APP=app.py
     ```
   - On Windows (PowerShell):
     ```sh
     $env:FLASK_APP="app.py"
     ```
   - On macOS/Linux:
     ```sh
     export FLASK_APP=app.py
     ```
2. Run the Flask application:
   ```sh
   flask run
   ```

### Step 6: PowerShell Execution Policy (If Needed)
If you're using **Windows Server 2008 R2**, you may need to adjust PowerShell execution policies.

1. Open **PowerShell as Administrator** and run:
   ```sh
   Set-ExecutionPolicy RemoteSigned
   ```
2. If you encounter a permission error, try:
   ```sh
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. After setup, you can revert it back with:
   ```sh
   Set-ExecutionPolicy Restricted
   ```

Now your Flask project is set up and ready to go!
